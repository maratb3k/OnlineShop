# 🛍️ OnlineShop
iOS маркетплейс-приложение, где покупатели могут просматривать и покупать товары, а продавцы — управлять своим ассортиментом.  
Приложение построено на **UIKit** с использованием **Firebase**.

---

## 🔐 Авторизация и онбординг

- Регистрация с выбором роли:
  - Покупатель
  - Продавец
- Вход по email и паролю через Firebase Auth
- Хранение профилей пользователей в Cloud Firestore
- Экран онбординга при первом запуске

---

## 🖼️ Работа с изображениями

- Загрузка фото товаров в Firebase Storage (`products/`)
- Загрузка аватарок в Firebase Storage (`avatars/`)
- Локальное кэширование аватарки
- Асинхронная загрузка изображений из URL

---

## 🏗️ Архитектура

| Слой | Детали |
|---|---|
| Паттерн | MVVM + Coordinator |
| UI | UIKit (программный UI без Storyboard) |
| Навигация | AppCoordinator с маршрутизацией по ролям через TabBar |
| Реактивность | Combine |
| Бэкенд | Firebase Auth, Cloud Firestore, Firebase Storage |
